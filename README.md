# Spotify-analysis
A meticulously curated compilation of 65K+ English songs spanning diverse genres, decades, and artistic expressions. This dataset offers an unparalleled opportunity for music aficionados, data enthusiasts, and researchers to immerse themselves in the rich tapestry of music.
Included Information:
•	Title of the Song
•	Artist Name
•	Spotify ID
•	Image
•	URL Link to the Song
Furthermore, the dataset's richness is amplified by harnessing the power of the Spotify API, which enables the extraction of Audio Features for each unique song. These encompass a comprehensive range of attributes, including:
•	Danceability: Describes how suitable a track is for dancing
•	Energy: Represents a perceptual measure of intensity and activity. Energetic tracks feel fast, loud and noisy.
•	Loudness: The overall loudness of a track in decibels (dB)
•	Speechiness: Detects the presence of spoken words in a track.
•	Acousticness: Describes whether a track uses only or primarily instruments that produce sound through acoustic means.
•	Instrumentalness: Predicts whether a track contains no vocals.
•	liveness: Detects the presence of an audience in the recording.
•	Valence: Describes the musical positiveness of a track
Within this dataset lie 20 columns brimming with information, each carefully curated and refined for seamless analysis. The potential for deriving illuminating insights from this dataset is boundless, offering a vast playground for profound explorations and meaningful discoveries.
